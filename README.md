# Formulário de Cadastro (Atividade programação web 3)

Formulário simples em **HTML, Bootstrap e PHP** para capturar e exibir dados do usuário.

## Como rodar

1. Abra o terminal na pasta do projeto.
2. Rode o servidor PHP embutido:

   ```bash
   php -S localhost:8000
   ```

   3.Acesse no navegador: http://localhost:8000/index.html

   Pronto! Preencha o formulário e clique em Salvar para ver os dados processados pelo PHP.
